LETTURA INFORMAZIONI MEDIA IMPORTATI

(Videomass User Guide)



Durante l’importazione di file multimediali, sulla finestra Gestione Presets o sulla finestra Video Edit, viene abilitato anche il pulsante Info Media, il quale avvia lo strumento ffprobe (un tool di ffmpeg) per la generazione di informazioni dal flusso multimediale, mostrandone il contenuto di dati. Su tali informazioni viene eseguito un parsing formattato in una finestra di due colonne: Parametri audio e Parametri video. Inoltre, un tab addizionale, denominato Informazioni estese, mostra lo stesso contenuto di dati ma in forma grezza, cioè privo di parsing.

Se un file importato è di tipo video/immagine, il riquadro Parametri video viene riempito con dei risultati. Se un file video presenta un flusso audio al suo interno, i dati relativi all’audio vengono aggiunti al riquadro Parametri audio. Se il file multimediale è solo di tipo audio, viene riempito solo il riquadro Parametri audio.

Queste informazioni possono venire utilizzate dagli utenti per definire anche il modo corretto di impostare i parametri di conversione.

NOTA: per utilizzare questa funzionalità, è necessario ffprobe. Vedi informazioni: http://www.ffmpeg.org/ffprobe.html
